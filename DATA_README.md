# Data 

This repository is a **portfolio project** and does not redistribute large or course-provided datasets.

To reproduce results:

1. Create a folder named `data/` in the repo root.
2. Place your dataset file as:
   - `data/game_teams_stats.csv`
3. If you do not have the full dataset, you can run the analysis with the included sample:
   - `data/game_teams_stats_sample.csv`

The analysis script automatically uses the full dataset if present, otherwise it falls back to the sample.
